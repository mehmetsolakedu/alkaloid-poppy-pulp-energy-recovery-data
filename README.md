# Dataset for torrefaction and fuel-quality assessment of alkaloid poppy pulp pellets

Version: 1.1
Date prepared: 2026-07-13
Public repository: https://github.com/mehmetsolakedu/alkaloid-poppy-pulp-energy-recovery-data

## Related manuscript

Torrefaction of alkaloid poppy pulp pellets: fuel-property gains constrained by ash and handling quality

## Description

This dataset supports the manuscript 'Torrefaction of alkaloid poppy pulp pellets: fuel-property gains constrained by ash and handling quality'. It contains local copies of the raw and processed analytical files used during manuscript audit and revision, including elemental analysis, ICP-OES, TGA/DTG, FTIR, XRF and related audit/provenance notes.

## Folder structure

- `raw_data/EA`: elemental / ultimate analysis files.
- `raw_data/ICP`: ICP-OES mineral and trace-element workbooks.
- `raw_data/TGA`: TGA/DTG analytical workbooks.
- `raw_data/FTIR`: FTIR archive files.
- `raw_data/XRF`: XRF archive files.
- `raw_data/reports`: analytical report files.
- `documentation`: raw-data provenance, replicate/statistical audit and correction notes.

## Use notes

The files are provided to support verification of the manuscript's descriptive analytical results. Complete replicate-level variance was not consistently available across every analytical domain; therefore, the associated manuscript interprets differences as descriptive measured changes and practical utilization boundaries rather than inferential statistical results.
